# Blog ETC Brasil Contabilidade

Este é o repositório oficial do código-fonte do Blog ETC Brasil Contabilidade.

O projeto é construído com:
- **Jekyll** como gerador de site estático.
- **Tailwind CSS** para a estilização.
- **Node.js** para o ambiente de desenvolvimento e compilação.

## Como rodar localmente

1. Clone o repositório.
2. Instale as dependências: `npm install`
3. Rode o servidor de desenvolvimento: `npm run dev`
